# gitmange
##Overview
gitmange is simple Task Tracker used to demonstarate Git and GitHub workflow scross four phases.

##Installation
1.Clone: git clone git@github.com<your-username>/gitmange.git
2.(Optional for Python) Craete venv: python -m venv venv && source venv/bin/activate
3. Run: python src/main.py

## Features
-Basic CLI demo app
-Demonstrates branching , merging and conflict resolution
-Github Actins CI workflow
-Documentation published using GitHub Pages

##Usage
Run python src/main.py to test a simple output
